# YT SEO Wizard — Web (Offline)
Double-click `index.html` to run. No install, no servers, works offline.

## How it works
1) Type a *topic* or paste *transcript text*.
2) Click **Generate**.
3) Download results as **.txt** or **.json**.

Outputs include:
- 10+ titles
- SEO description (600–1600 chars)
- Tags line (comma-separated)

Supports EN / RU. All logic runs in your browser.
